# tieba-dl

    An image crawler for [BaiduTieba](https://tieba.baidu.com/)

----

# Quick Start
  `pip3 install -r requirement.txt`
  edit your watch list file 'tieba.list'
  `make`

# User Setting File
  - [Watchlist](/tieba.list), manaully create one if not exists, one url per line
